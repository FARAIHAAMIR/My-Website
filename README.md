Built My First Mini E-commerce Store LuxeShop Using HTML & Inline CSS! 👗🛍️

I created a simple yet functional fashion store webpage from scratch using HTML and inline CSS. This project helped me understand how tags, attributes, and elements work together to display images, descriptions, and links neatly.
---
💡 What I practiced:

Structuring content with <html>, <body>, and <title>

Displaying product images using <img>

Organizing text with <p> and headings <h1>/<h2>

Adding clickable links with <a>

Styling elements inline to arrange items neatly

🌟 Key takeaway:

HTML alone is powerful enough to build clean and organized layouts. Learning its core structure makes it easier to integrate CSS and interactive features later.
